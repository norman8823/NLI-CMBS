# Surveillance Report Quality Checklist

## Terminology (Must Pass All)
- [ ] Uses "UPB" not "unpaid principal balance" or "balance" alone
- [ ] Uses "specially serviced" not "troubled" or "distressed"
- [ ] Uses "DSCR" (spelled out only on first use if at all)
- [ ] Uses "WA" for weighted average
- [ ] Property types use standard codes or full names (MF, RT, OF, IN, LO, MU, HC, MH, SS, WH)

## Numerical Formatting (Must Pass All)
- [ ] Dollar amounts: $X.XM or $X.XXB format
- [ ] Percentages: one decimal place (4.5%)
- [ ] DSCR: two decimals with x suffix (1.45x)
- [ ] Dates: Month YYYY format

## Structure (Must Pass All)
- [ ] All 6 sections present in order:
  1. Executive Summary
  2. Deal Performance Overview
  3. Delinquency & Special Servicing
  4. Maturity & Refinancing Risk
  5. Top Loans
  6. Outlook
- [ ] Executive Summary is 3-4 sentences
- [ ] Data Source footer with filing date and accession number
- [ ] EDGAR URL present

## Content Quality (Must Pass All)
- [ ] Top loans section names specific properties with locations
- [ ] Delinquency section lists actual loans if any are delinquent
- [ ] Maturity section references specific years and amounts
- [ ] No fabricated loan names or metrics

## Red Flags (Must NOT Appear)
- [ ] No "I" or "we"
- [ ] No "This report" or "this analysis"
- [ ] No "based on the data provided" or "as shown above"
- [ ] No "in conclusion" or "to summarize"
